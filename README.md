## Concert Seat Map 
A map of seats at a concert venue. Stores data for each seat's sold status, ticket type (Regular, Premium, or General Admission), and price. Each seat is accessible and can be changed within the venue.

## Matrix Puzzle Board 
Generates a random 5x5 board with integers from 1-15. The board can then be checked for adjacent pairs and duplicates of all values. Additionally, rows can be shifted right, columns can be shifted up, and rows/columns can be reversed.
